
## Веб-интерфейс дашборда

Проект содержит полнофункциональный веб-интерфейс дашборда для управления балансом:

### Страницы дашборда

#### 1. Авторизация (`/login`)
- Форма входа в систему
- Валидация учетных данных
- Защита от CSRF атак
- "Запомнить меня" функционал

#### 2. Главная страница (`/dashboard`)
- Отображение текущего баланса
- Статистика по операциям (начислено/списано)
- Панель быстрых действий
- Адаптивный дизайн

#### 3. История операций (`/dashboard/history`)
- Таблица всех операций пользователя
- Фильтрация по типам и статусам
- Пагинация результатов
- Детальная информация по каждой операции

### Навигация и меню

Верхнее меню содержит:
- **Логотип и название системы**: ссылка на главную
- **Главная**: текущий баланс и статистика
- **История операций**: полный список транзакций
- **Выход**: завершение сессии

### Vue.js интеграция

На всех страницах подключен Vue.js v3:
- Компонент "Vue.js Example" в правом нижнем углу
- Демонстрация работы Vue методов
- Alert с текстом "test123" при клике

### Bootstrap и стилизация

Используется Tailwind CSS:
- Современный адаптивный дизайн
- Цветовые индикаторы операций
- Hover эффекты и анимации
- Semantic HTML структура

### Права доступа и безопасность

- Middleware проверки аутентификации
- Защита от неавторизованного доступа
- Безпечное хранение сессий
- CSRF токены на формах

### Демо данные

Для тестирования создайте пользователя:
```bash
php artisan user:create --name="Демо Пользователь" --email="demo@example.com" --password="demo123" --with-balance
php artisan user:balance-operation --user="demo@example.com" --type="deposit" --amount="1000" --description="Демо пополнение"
```

Затем войдите в систему по адресу `/login` с учетными данными `demo@example.com` / `demo123`.
